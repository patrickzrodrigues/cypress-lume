# My Cypress Automation

Este projeto utiliza o Cypress para automatizar os teste de um recurso de login e um recurso de atuaização de usuário. O Cypress é um estrutura de testes ponto a ponto (E2E) baseada em JavaScrip que permite oas desenvolvedores escrever testes e executá-los em um navegador real.

## Instalação
```bash
npm install
```
>**NOTE;**
>
> é necessáro ter o **node** instalado

## Executar
```bash
# Você deve abrir a interface do Cypress
npx cypresss open

# Execute a atumação pelo linha de comando
npx cyress run
```
